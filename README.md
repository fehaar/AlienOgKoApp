# AlienOgKo

A mobile game built with Unity.

## Install on Android

Download the latest APK from the [Releases](https://github.com/fehaar/AlienOgKoApp/releases/latest) page and install it on your Android device.

> Make sure **"Install from unknown sources"** is enabled on your phone (Settings → Security → Install unknown apps).

## How to play

- **Shake the phone upward** to trigger the bounce animation
- Keep the rhythm going to loop the animation
- Let the phone rest to play out the full sequence
